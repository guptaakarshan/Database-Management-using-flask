# Flask Authentication System

A secure user authentication and grade management system built using:

- Flask
- MySQL
- Flask-Bcrypt
- Bootstrap

## Features
- User Registration
- Login & Logout
- Profile Update
- Password Reset
- Grade Viewing
- Responsive UI

## Setup Instructions

1. Install dependencies:
  pip install flask flask-mysqldb flask-bcrypt flask-login python-dotenv
2. Setup MySQL database

3. Create `.env` file with SECRET_KEY

4. Run: python app.py
